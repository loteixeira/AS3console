Classpublic class ConsoleCore
InheritanceConsoleCore Object

AS3console component main class, where you can write/read data, throw/listen events, etc. You can use AS3console through Console class, where a single instance may be accessed from any point of the code, or creating an instance of ConsoleCore class. If you instantiate this class you must keep a reference to the object and cpln don't work (this alias is tied to Console static instance). Using instances of this class instead of Console single instance allows you to manage several different consoles. Console embed commands: There are some embed input commands that you can type in console window:
  • alpha [value]: get console component alpha value, if value exists set console alpha.
  • clear: clear console text area.
  • hide: hide console window.
  • mem: show memory used by the flash player.
  • plugins: list installed plugins.
  • version: print console version details.
    • Everytime text is inserted into console input field ConsoleEvent.INPUT event is thrown, if the text isn't a embed command.

    public function ConsoleCore(parent:DisplayObjectContainer = null, assetFactory:AssetFactory = null, eventDispatcher:IEventDispatcher = null)

    Create an instance of AS3console component. If parent is non-null the instance will have a graphical interface, otherwise it runs in release mode (only throwing events and responding to config.traceEcho and config.jsEcho). If assetFactory is null, a instance of DefaultAssetFactory is created. If you want a single instance through static Console class, see Console.create() method. Note: parent object MUST be part of the display list in such way that parent.stage attribute is valid.

    Parameters
    parent:DisplayObjectContainer (default = null) — An instance of DisplayObjectContainer to add console component as child. If this value is null, console runs in release mode.
     
    assetFactory:AssetFactory (default = null) — Use to specify a non-default instance of AssetFactory.
     
    eventDispatcher:IEventDispatcher (default = null) — An object to dispatch ConsoleCore events.
