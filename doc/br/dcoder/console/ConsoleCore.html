Packagebr.dcoder.console
Classpublic class ConsoleCore

AS3console component main class, where you can write/read data, throw/listen events, etc. You can use AS3console through Console class, where a single instance may be accessed from any point of the code, or creating a instance of the ConsoleCore class. Whether you instantiate this class, you must keep a reference to the object. However, cpln won't work (the alias is tied to Console static instance). Using instances of this class instead of Console single instance allows you to manage several different consoles.

Constructor detail
ConsoleCore()constructor
public function ConsoleCore(parent:DisplayObjectContainer = null, assetFactory:AssetFactory = null, eventDispatcher:IEventDispatcher = null)

Create an instance of AS3console component. If parent is non-null the instance will have a graphical interface, otherwise it runs in release mode. If assetFactory is null, a DefaultAssetFactory is used.
Note: parent object MUST be part of the display list in such way that parent.stage attribute is valid.

Parameters
parent:DisplayObjectContainer (default = null) — DisplayObjectContainer to add console component. If this value is null, console runs in release mode.
 
assetFactory:AssetFactory (default = null) — Use to specify a non-default instance of AssetFactory.
 
eventDispatcher:IEventDispatcher (default = null) — An object to dispatch ConsoleCore events.

public function shortcut(shortcutKeys:Array, shortcutUseAlt:Boolean = false, shortcutUseCtrl:Boolean = true, shortcutUseShift:Boolean = false):void

Set show/hide shortcut. If this shortcut is already used by container application (Flash IDE, Web Browser, etc) or running release mode, console shortcut will never be triggered.

Parameters
shortcutKeys:Array — An array with all keys that should be pressed to trigger shortcut. Each key is a one-character string. Example: [ "m" ].
 
shortcutUseAlt:Boolean (default = false) — If true ALT key should be pressed to trigger shortcut.
 
shortcutUseCtrl:Boolean (default = true) — If true CTRL key should be pressed to trigger shortcut.
 
shortcutUseShift:Boolean (default = false) — If true SHIFT key should be pressed to trigger shortcut.
