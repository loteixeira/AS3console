Packagebr.dcoder.console.plugin
Classpublic class ConsolePlugin
InheritanceConsolePlugin Object
Subclasses CodeEval, LocalClient, LocalServer

AS3console plugins base class. A plugin can listen for console input and output. To add a plugin to a ConsoleCore instance use installPlugin method. An instance of a plugin MUST be installed in only one ConsoleCore object.

See also
