CodeEval()constructor
public function CodeEval(_context:Object = null)

Create an instance of CodeEval plugin. Using this plugin every time you type "eval [source code]" in console input text field, [source code] block will be compiled to AVM2 bytecode and executed. Generated byte code runs from inside a function, where receive two parameters:

  • getDef:Function -> alias for flash.utils.getDefinitionByName
  • context:Object -> context object

    • Parameters
    _context:Object (default = null) — An acessible oobject for evaluated source code
