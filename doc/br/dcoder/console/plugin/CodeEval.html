start()method
public static function start(_context:Object = null):void

Start CodeEval plugin. Using this plugin every time you type "eval [source code]" on console input text field, [source code] block will be compiled to AVM2 bytecode and executed. Generated byte code runs from inside a function, where receive two parameters:

  • getDefinition:Function -> alias for flash.utils.getDefinitionByName
  • context:Object -> context object

    • Parameters
    _context:Object (default = null) — An acessible object for evaluated source code
