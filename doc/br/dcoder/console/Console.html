Classpublic class Console
InheritanceConsole Object

Wrapper of a single ConsoleCore instance, used to simplify logging process. You can use AS3console through this class, accessible from any point of your code, or creating and managing instances of ConsoleCore separately. You shouldn't instantiate this class using regular constructor, instead use create static method.

public static function create(stage:Stage = null, assetFactory:AssetFactory = null, eventDispatcher:IEventDispatcher = null):ConsoleCore

Create a ConsoleCore instance to be wrapped within Console static class. This method must be called once. If stage instance if null, ConsoleCore instance runs in release mode, where it dispatches all events but has no graphical interface.

Parameters

stage:Stage (default = null) — Reference to application stage object.
 
assetFactory:AssetFactory (default = null) — Use to specify a non-default instance of AssetFactory.
 
eventDispatcher:IEventDispatcher (default = null) — An object to dispatch ConsoleCore events.

Returns
ConsoleCore — ConsoleCore instance.
